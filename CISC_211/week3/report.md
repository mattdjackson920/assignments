# Week 3 - Variables and Constants

## Objective

The objective of this assignment was to learn how to declare and use initialized and uninitialized variables in Assembly language. The program assigns values to two variables, adds them together, and stores the result in another variable.

## Challenges Encountered

One challenge was understanding the difference between initialized variables in the `.data` section and uninitialized variables in the `.bss` section. Another challenge was learning how to move values between memory and registers before performing arithmetic operations.

## Program Explanation

The program initializes two variables, `var1` and `var2`, with the values 10 and 15. Their values are loaded into the CPU, added together, and the result is stored in the uninitialized variable `result`.

The program then exits normally.
